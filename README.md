# شركة الحمرا سي آي - الموقع الإلكتروني

موقع شركة الحمرا سي آي للمواد العازلة والإنشائية - تصميم حديث واحترافي.

## التشغيل

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

الملفات الجاهزة للنشر ستكون في مجلد `dist`.
